# 🚗 Car Price Prediction AI

This is a Machine Learning project that predicts car selling prices based on features like mileage, age, fuel type, and more.

---

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

---

##  Features
- Predict car price using ML model
- Simple and interactive UI
- Real-time input
- Clean user experience
